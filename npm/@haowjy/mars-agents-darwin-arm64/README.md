# @haowjy/mars-agents-darwin-arm64

Platform binary package for `mars` on macOS `arm64`.

This package is normally installed automatically as an optional dependency of `@haowjy/mars-agents`. Most users should install the top-level package instead:

```bash
npm install -g @haowjy/mars-agents
```
